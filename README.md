# Zols Console Expense Tracker

A console-based expense tracking application built with C# and .NET.

## Features
- Add expenses
- View expenses
- Calculate total expenses
- Input validation

## New Features
- Category Filtering
- Expense Categories
- Delete Expense Feature
- Expense Validation
- Montly Expense Calculations

## Technologies
- C#
- .NET 10
- Console Application

## Project Structure
- Models
- Services
- Storage
